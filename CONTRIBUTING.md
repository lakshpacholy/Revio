# Contributing

## Branch naming

```
issue-<number>-<short-description>
```

## Commit messages

Reference the issue number in every commit:

```
Set up X (#1)
Add Y and Z (#4)
Fix broken import in config (#7)
```

## Pull requests

- Use `Closes #<number>` in the PR body to auto-close the linked issue on merge.
- Fill in the PR template — especially the testing section.
